    • Explicación

    La presente práctica consistió en realizar una aplicación web para leer y procesar ficheros ini . Para su ejecución, se hizo uso de los conocimientos adquiridos en prácticas anteriores:

    • Desarrollo de la app con los lenguajes HTML, CSS y Javascript.
    • Uso de las librerías JQuery (para el manejor de eventos) y Underscore.js (para estructurar la salida).
    • Uso de LocalStorage y SessionStorage y la API de HTML5 en JavaScript que lo maneja.
    • Entorno de pruebas basado en Mocha y Chai.
    • JSON (JavaScript Object Notation), concretamente el método stringify() para añadir soporte de multilíneas en las asignaciones.
    Adicionalmente, para esta práctica es necesario el uso de:
    • La API de HTML5, File API, para el manejo de ficheros locales.

    Práctica 3 : INI File Parser

    INSTRUCCIONES DE USO

    Los ficheros ini a procesar pueden arrastrarse al área señalada o subir un fichero buscando con el explorador de archivos en su ordenador. Una vez, subido o arrastrado, se procesa el fichero y aparecerá parseado en el parte inferior, el cual se divide en dos columnas: en la izquierda se verá el fichero original; y en la derecha el árbol sintáctico.

    • input2.ini
